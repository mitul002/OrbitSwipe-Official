**OrbitSwipe** is a state-of-the-art radial launcher and workspace manager for Windows 10 & 11. Designed to bring a premium, customizable glass UI to your desktop, OrbitSwipe allows you to launch apps, manage windows, control system media, and view system stats with mouse-gesture precision.

---

## ✨ Key Features

### 1. 🌀 Radial Launcher with Glass UI
* Summon a customizable, fully hardware-accelerated radial menu at your mouse cursor. Fast, fluid, and beautifully designed.

### 2. 🪟 WinTop Window Manager
* Easily toggle windows to stay always-on-top, manage window transparencies, and snap layouts seamlessly.

### 3. 📊 Live System Stats
* Real-time monitoring of CPU, RAM, Network speeds, and disk reads embedded directly in your launcher interface.

### 4. 🎛️ Volume & Brightness Sliders
* Quick-action mouse wheels and radial sliders to adjust audio levels and display brightness with zero hassle.

### 5. 🔍 Smart Global Search
* Quickly search files, launch applications, search the web, and execute shortcut commands from a unified bar.

### 6. 🎨 Custom Glass Accents & Themes
* Fully customize the radius, blur strength, acrylic opacity, and neon glass glow parameters to match your wallpaper or desktop setup.

## 💾 Installation

### Method 1: Direct Download (Recommended)
Get the latest setup installer directly from our **[Releases Page](https://github.com/mitul002/OrbitSwipe-Official/releases/latest)**.

### Method 2: Winget
Once indexed, install OrbitSwipe via Windows Package Manager:
```cmd
winget install OrbitSwipe
```

### Method 3: Chocolatey
Once approved, install via Chocolatey:
```cmd
choco install orbitswipe
```
